**Badge** — small status pill for appointment state, insurer acceptance, or distinctions.

```jsx
<Badge tone="success" dot>Confirmed</Badge>
<Badge tone="accent">EndoArt Pioneer</Badge>
```

Tones `neutral | primary | success | warning | danger | accent`; `dot` adds a status dot; `solid` for a filled high-emphasis pill; `size="sm"` for dense rows.
