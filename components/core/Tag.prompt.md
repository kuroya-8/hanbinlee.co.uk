**Tag** — pill for specialties, conditions treated, and filter chips.

```jsx
<Tag variant="soft">Cataract surgery</Tag>
<Tag variant="outline" onRemove={clear}>Corneal</Tag>
```

Variants `soft` (teal tint, default) · `outline` · `neutral`. Pass `iconLeft` for a leading icon, `onRemove` to make it dismissible.
